# User Guide

Welcome to the Graphy user guide.

## Installation

Install via cargo:

```bash
cargo install graphy
```

## Usage

Run the pipeline:

```bash
graphy run --root ./src
```

### Basic Options

Use `--help` for a full list of options.

### Advanced Configuration

See [API Reference](api.md) for advanced usage.

## Troubleshooting

Common issues and solutions.

### Performance

If the pipeline is slow, try disabling hierarchical clustering.
